# Payroll System

in this time we make payroll system based on web application.

## Features

- Light/dark mode toggle
- Responsive layout
- Fullscreen mode
- Cross platform
- Employee payroll
- Verification for new Employee
- Payroll history of Employee
- Statistic of Employee
